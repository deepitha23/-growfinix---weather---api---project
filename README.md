# 🌤 Weather API Project (Python)

## 📌 About the Project
This is a simple Python project that fetches real-time weather data for any city using the WeatherAPI service. It displays temperature, weather condition, and wind speed in a clean command-line interface.

---

## 🚀 Features
- Get real-time weather updates
- City-based search input
- Displays temperature in Celsius
- Shows weather condition
- Shows wind speed
- Simple and beginner-friendly Python project

---

## 🛠️ Technologies Used
- Python
- Requests library
- WeatherAPI (External API)
